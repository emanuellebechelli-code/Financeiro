# Financeiro

App local de planejamento financeiro pessoal.

## Como usar

Abra `index.html` no navegador. O app salva os dados no próprio navegador usando armazenamento local.

## Recursos

- saldo inicial;
- entradas e gastos diários;
- assinaturas separadas;
- cálculo de dízimo sobre entradas;
- gráficos em pizza;
- metas em `Preciso adquirir`.
